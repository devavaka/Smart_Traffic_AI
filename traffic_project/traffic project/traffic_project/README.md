# 🚦 Vehicle Tracking & Traffic Analytics System

A real-time traffic monitoring desktop application built with **Python**, **YOLOv8**, **OpenCV**, and **Tkinter**.

Designed as a final-year college project — fully functional, CPU-compatible, and demo-ready.

---

## 📸 Features

| Feature | Description |
|---|---|
| **Vehicle Detection** | YOLOv8n (nano) detects Cars, Bikes, Trucks & Buses |
| **Multi-object Tracking** | Centroid IoU tracker assigns persistent IDs across frames |
| **Vehicle Counting** | Cumulative count + real-time Vehicles Per Minute (VPM) |
| **Accident Prediction** | Rule-based risk score using speed, proximity & convergence |
| **Heatmap Overlay** | Gaussian accumulation heatmap shows traffic density |
| **Bird's Eye View** | Perspective transform to top-down road view |
| **Number Plate Detection** | ROI extraction + synthetic plate display (extendable with OCR) |
| **Live Dashboard** | Dark-themed Tkinter GUI with real-time analytics panels |
| **Webcam Support** | Use any connected USB/built-in camera |
| **Video File Support** | Upload and analyse any `.mp4 / .avi / .mov / .mkv` |
| **Mock Mode** | If YOLOv8 is not available, animated synthetic detections keep the demo running |

## ⚙️ Setup Instructions

### 1. Prerequisites

- Python **3.10** or newer
- pip (bundled with Python)
- Windows / macOS / Linux

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

> **First run note:** Ultralytics will automatically download `yolov8n.pt` (~6 MB) on first detection. An internet connection is required once.

### 4. Generate the sample video

```bash
python generate_sample.py
```

This creates `assets/sample_video.mp4` — a synthetic traffic video you can use immediately for demo.

## ▶️ How to Run

```bash
python main.py
```

The dashboard window opens. Then:

1. **Upload a video** — click `📂 Upload Video` and select any video file.  
   *or*  
   **Start webcam** — click `🎥 Start Webcam` to use your camera.

2. Click **▶ Start Analysis** to begin detection, tracking & analytics.

3. Toggle **Show Heatmap Overlay** (checkbox below video) to see the movement heatmap.

4. Click **■ Stop Analysis** → **↺ Reset** between sessions.

## 🔧 Configuration

Edit constants at the top of each module to tune behaviour:

| File | Constant | Default | Effect |
|---|---|---|---|
| `detection/yolo_model.py` | `CONF_THRESHOLD` | `0.40` | Detection confidence cutoff |
| `detection/tracker.py` | `IOU_THRESHOLD` | `0.30` | Matching strictness |
| `detection/tracker.py` | `MAX_MISSING` | `15` | Frames before track dropped |
| `analytics/accident.py` | `HIGH_SPEED_PX` | `25.0` | Speed threshold (px/frame) |
| `analytics/accident.py` | `MIN_SAFE_DIST_PX` | `80.0` | Proximity danger zone |

## 🚀 Extending the Project

### Real OCR for number plates
```bash
pip install easyocr
```
Replace `_synthetic_plate()` in `detection/yolo_model.py` with:
```python
import easyocr
reader = easyocr.Reader(["en"])
result = reader.readtext(plate_crop, detail=0)
return result[0] if result else None
```

### GPU acceleration
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
YOLO will automatically use CUDA if available.

### DeepSORT tracking (stronger re-ID)
```bash
pip install deep-sort-realtime
```
Replace `VehicleTracker` usage in `dashboard.py` with `DeepSort`.
